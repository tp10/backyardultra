Backyard Ultra Wales Basics

Each loop is 4 miles 880 feet or 6.7056 kilometers in length. All competitors must start at the bell (no late starts). Each loop starts precisely 1 hour after the last.

Easy Running

Every hour you will need to complete a 4.2 mile lap. If you get back from the lap in 40 minutes, you will have 20 minutes rest before starting the next lap. Laps are not timed, you just need to get back from each lap within the hour.

Run from 9am until you can run no more or you're the last one standing

The uncapped lap event we're running will take place from 9am until the last person is standing. If you're doing well you'll be running into the night.

Narberth Nobbler

The event is run by the Narberth Nobbler charity.
All profits go to Welsh charities.
